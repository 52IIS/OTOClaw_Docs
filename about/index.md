# 关于我们

## 项目简介

OTOClaw 一键龙虾是一个开源的 OpenClaw 一键安装整合包，基于 Tauri 2.0 + Vue 3 + TypeScript + Rust 构建的跨平台 AI 助手管理工具。我们的目标是让用户能够轻松、快速地部署和使用龙虾 AI 系统。

## 我们的使命

- **简化部署**：让复杂的 AI 系统部署变得简单易行
- **开源共享**：所有代码开源，欢迎社区贡献
- **持续迭代**：不断优化用户体验，增加新功能
- **用户至上**：倾听用户反馈，持续改进产品

## 技术栈

| 技术 | 说明 |
|------|------|
| Tauri 2.0 | 跨平台桌面应用框架 |
| Vue 3 | 渐进式 JavaScript 框架 |
| TypeScript | 类型安全的 JavaScript 超集 |
| Rust | 高性能系统编程语言 |

## 反馈指引

在提交问题反馈前，请准备好以下信息，这将帮助我们更快地定位和解决问题：

### 必要信息

1. **系统环境**
   - 操作系统及版本（如 Windows 11、macOS 14、Ubuntu 22.04）
   - OTOClaw 版本号

2. **问题描述**
   - 详细描述遇到的问题
   - 问题复现步骤
   - 期望的行为 vs 实际的行为

3. **错误信息**
   - 截图或错误日志
   - 控制台输出（如有）

### 反馈模板

```
**系统环境：**
- 操作系统：[如 Windows 11]
- OTOClaw 版本：[如 1.0.0]

**问题描述：**
[详细描述问题]

**复现步骤：**
1. [步骤一]
2. [步骤二]
3. [步骤三]

**期望结果：**
[描述期望的行为]

**实际结果：**
[描述实际发生的情况]

**截图/日志：**
[附上相关截图或日志]
```
